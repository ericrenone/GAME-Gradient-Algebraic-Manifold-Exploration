## II.2 The Permeability Alphabet and Its Ordering

**Definition (Permeability Alphabet).** The set of all reduced gradient-ratio fractions:

```

```
(p₁,q₁) ≼_𝒜 (p₂,q₂)  iff  q₁ ≤ q₂

**Theorem (𝒜 is SP).** The map (p,q) ↦ q is an order-isomorphism from (𝒜, ≼_𝒜) to (ℕ, ≤). Since (ℕ, ≤) is SP, so is (𝒜, ≼_𝒜). ∎

**Why denominator-only ordering?** The denominator ordering is the *unique* total preorder on 𝒜 making (𝒜, ≼_𝒜) isomorphic to (ℕ, ≤), given three constraints: (i) invariance under Farey mediant operations, (ii) Ford-circle radius interpretation 1/(2q²), and (iii) monotonicity in generalization. Any ordering incorporating numerators produces a partial order admitting infinite antichains.

This pair satisfies both coordinates. By induction, (ℕᵈ, ≼_componentwise) is SP for all finite d, with ordinal bound o*(ℕᵈ) = ωᵈ. ∎

## II.5 The Sequence Anchor (Higman's Lift)

**Theorem (Sequence Anchor).** Let (A, ≼_A) be SP. Define subsequence dominance ≼* on finite sequences A* by: u ≼* v iff there exist indices i₁ < i₂ < ··· < iₖ with u = u₁···uₖ and uⱼ ≼_A v_{iⱼ} for all j. Then (A*, ≼*) is SP.

**Proof (minimal Resistance Chain construction).** Assume for contradiction that (A*, ≼*) is not SP. Choose a minimal Resistance Chain B = (w₀, w₁, ...) in (A*, ≼*): w₀ is ≼_A-minimal among first elements of infinite Resistance Chains; each subsequent wₙ is minimal given the prior choices. Extract first letters aᵢ := first(wᵢ). Since A is SP, there exist i < j with aᵢ ≼_A aⱼ. Remove the first letter from wᵢ to form wᵢ'. The resulting sequence is a strictly smaller Resistance Chain — contradicting minimality of B. ∎

**Finite Alphabet Condition:** The full alphabet 𝒜 is countably infinite but **effectively finite within any bounded resolution**: for fixed Q, the sub-alphabet 𝒜_Q := {(p,q) ∈ 𝒜 : q ≤ Q} has exactly Σ_{k=1}^Q φ(k) elements (Euler totient sum), which is finite. All training trajectories operate within 𝒜_Q for Q_max = ⌊1/ε_min⌋.

## II.6 The Recursive Tree Anchor (Kruskal's Theorem)

**Definition (Trajectory Tree).** The trajectory tree T_t is the finite rooted tree whose nodes are training states s_τ for τ ≤ t, labeled by Permeability Convergents (p_τ, q_τ) ∈ 𝒜, with branching corresponding to q*-decreasing (convergence) or q*-increasing (resistance) steps.

**Definition (Homeomorphic Permeation Order).** T ≼_T T' if there exists an injective node-map φ: nodes(T) → nodes(T') such that: (i) label(φ(v)) ≽_𝒜 label(v) for each node v; (ii) u is ancestor of v implies φ(u) is ancestor of φ(v); (iii) sibling order is preserved.

**Theorem (Recursive Tree Anchor — Kruskal 1960).** Let (A, ≼_A) be SP. The set 𝒯(A) of finite rooted A-labeled trees under ≼_T is SP, with ordinal bound o*(𝒯(ℕ²)) = ε₀.

**Proof structure:** Assume (𝒯(A), ≼_T) is not SP. Construct a minimal Resistance Chain of trees by the same minimality argument as the Sequence Anchor, applied to trees. The key lemma (Kruskal): for any tree T in a minimal Resistance Chain, its sub-forests form a sequence that is SP by the Sequence Anchor. This creates a contradiction — the sub-forests cannot form an infinite Resistance Chain. ∎

**Implication:** Every Resistance Chain of trajectory trees has ordinal length strictly less than ε₀ = ω^{ω^{ω^{···}}} (the Feferman-Schütte ordinal). This is the theoretical ceiling for all finite gradient-tree configurations.
